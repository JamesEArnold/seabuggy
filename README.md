# SeaBuggy
SeaBuggy is an OpenSea bot.

## Installation

Use [npm](https://www.npmjs.com/) to install SeaBuggy.

```bash
npm install
```

## Usage

Start SeaBuggy using the following npm command:

```bash
npm run dev
```
